# DMM_Hide_Launcher
一款简洁且shit的DMM游戏启动器

## 📋 项目简介

DMM_Hide_Launcher 是一款专为DMM游戏打造的启动器，旨在简化游戏启动流程、优化账号管理体验，并提供版本检查功能，让玩家能够更便捷地享受游戏乐趣

## ✨ 核心功能

- **快速启动**：一键启动游戏，跳过繁琐的登录过程
- **多账号管理**：安全保存多个游戏账号，支持快速切换
- **自动版本检查**：实时获取游戏更新信息和版本状态
- **自定义配置**：灵活设置游戏路径和平台参数
- **详细日志系统**：记录操作过程，方便排查问题
- **现代用户界面**：基于AdonisUI的简洁美观界面设计
- **智能通知系统**：只在激活窗口显示信息通知，避免UI挤压和重叠
- **暗色/亮色主题**：支持主题切换，适应不同使用环境

## 📖 使用指南

### 首次运行配置
1. 运行程序后，系统会自动创建默认配置文件 `config.xml`
2. 在主界面点击 "游戏路径" 输入框，手动输入游戏安装路径
   ```
   例如，若游戏可执行文件路径为 `C:\dmmdzz_4399\game\dmmdzz.exe`，则应写入路径 `C:\dmmdzz_4399`
   ```
   或点击 "游戏路径" 输入框右侧的 "自动查找" 按钮，自动查找游戏安装目录
3. 程序会自动检测已安装的游戏路径，并提供多个路径供选择（如果找到多个）

### 账号管理
1. 在主界面点击"账号管理"按钮，打开账号编辑窗口
2. 输入用户名和密码，点击"保存"按钮添加新账号
3. 对于已添加的账号，您可以进行编辑或删除操作
4. 支持账号导入导出功能，方便在不同设备间迁移账号信息
5. 所有账号信息会使用AES加密算法加密后保存在本地的 `accounts.json` 文件中
   > 注意：尽管程序使用了加密技术保护您的账号信息，但仍请妥善保管您的程序和配置文件，避免泄露


### 启动游戏
1. 在主界面的账号列表中选择您要使用的账号
2. 点击"启动游戏"按钮，程序会自动处理登录和启动流程
3. 等待片刻，游戏将会自动启动

## ⚙️ 配置详解

### 配置文件 (config.xml)
配置文件位于程序根目录，采用XML格式，包含以下主要配置项：

```xml
<?xml version="1.0" encoding="utf-8"?>
<config>
  <game_data>游戏安装路径</game_data>
  <ID_7K>平台ID</ID_7K>
</config>
```

- **game_data**：游戏客户端的安装路径
- **ID_7K**：游戏平台的唯一标识ID

> 💡 提示：您可以直接编辑此文件或通过程序界面进行配置修改

### 信息通知系统
- 程序采用智能通知系统，只在当前激活的窗口显示通知信息
- 通知面板采用浮动设计，不会占用其他界面元素空间
- 在主窗口和账号编辑窗口均有独立的通知系统，确保信息传递及时且不干扰用户操作

## 🤔 常见问题与解决方案

### 游戏无法启动
- **问题**：点击"启动游戏"后无反应或提示错误
- **解决方法**：
  1. 检查配置文件中的游戏路径是否正确
  2. 确认游戏客户端已正确安装
  3. 查看日志文件获取详细错误信息

### 账号无法保存或加载
- **问题**：添加账号后无法保存或重启程序后账号丢失
- **解决方法**：
  1. 确保程序有文件写入权限（尝试以管理员身份运行）
  2. 检查 `accounts.json` 文件是否损坏，可尝试删除后重新创建

### 网络连接问题
- **问题**：无法获取游戏版本信息
- **解决方法**：
  1. 检查网络连接是否正常
  2. 确认防火墙设置没有阻止程序访问网络
  3. 离线状态下程序仍可启动游戏，但无法获取更新信息
